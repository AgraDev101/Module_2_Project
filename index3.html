INCLUDE CLASS AS ATTRIBUTE IN TAG TO ADD STYLE TO THIS TAG

In CSS file to access class use ".class-name" then curly bracket "{ }" and then mention style propeties inside the curly bracket. If you don't want class just use tag name to target that HTML element, but remember styles given by targeting HTML tag will apply on all HTML tags of same name.

To Target this element. .section-4 > div > h3 { your CSS properties }

To style this paragraph in your CSS file. because section-4 class is inside section tag we need to write for example .section-4 > div > p { color: "black"; }
